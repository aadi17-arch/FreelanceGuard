# 🏛️ FreelanceGuard | Secure Payments for Freelancers

**"Peace of mind for every project, every payment, every time."**

FreelanceGuard is a premium platform designed to make freelance payments safe and simple. We provide a secure space where freelancers and clients can work together without worrying about payment uncertainty.

---

## ✨ Why FreelanceGuard?
We believe that professional work deserves professional protection. Our platform is built on three main pillars:
- **Simplicity**: We use clear, everyday language so you always know where your money is.
- **Safety**: Your funds are held securely until the work is finished and approved.
- **Style**: A clean, modern interface that looks as professional as your business.

---

## 🚀 Built With
- **Frontend**: React 19 & Tailwind CSS (for a fast, beautiful experience)
- **Backend**: Node.js & Express (for reliable performance)
- **Database**: Prisma & PostgreSQL (for secure data management)
- **Security**: Encrypted sessions and strict data validation.

---

## 🛠️ Key Features

### 1. Payment Protection
- **Milestone Payments**: Break your projects into smaller steps and pay as you go.
- **Secure Vault**: Funds are locked safely before work begins, protecting both parties.
- **Easy Withdrawals**: Move your earnings to your bank account with one click.

### 2. My Workspace
- **Personal Dashboard**: A clean overview of your earnings, active projects, and safety balance.
- **Professional Profiles**: Show off your work history and verification status.
- **Activity History**: A complete, easy-to-read record of every payment.

### 3. Support & Safety
- **Conflict Resolution**: A dedicated space to handle project disagreements fairly.
- **Verified Users**: Look for the badge to know you're working with a trusted partner.
- **Help Center**: Simple guides to help you get the most out of the platform.

---

## 🚦 Installation Guide

### 1. Clone the Project
```bash
git clone https://github.com/aadi17-arch/FreelanceGuard.git
cd freelanceguard
```

### 2. Start the Server
```bash
cd server
npm install
npx prisma generate
npm run dev
```

### 3. Start the Website
```bash
cd client
npm install
npm run dev
```
