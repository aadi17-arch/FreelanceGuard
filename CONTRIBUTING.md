# Contributing to FreelanceGuard

Thank you for your interest in contributing to **FreelanceGuard**! We welcome developers who want to help build the most secure and atmospheric freelance platform.

## 🛠️ Development Workflow
1. **Fork** the repository and create your branch from `main`.
2. **Setup** the environment using the instructions in the [README](README.md).
3. **Write** clean, semantic code following the existing design system (cinematic glassmorphism).
4. **Test** your changes thoroughly.
5. **Submit** a Pull Request with a detailed description of your updates.

## 🎨 Design Guidelines
- Use **Framer Motion** for all transitions.
- Maintain the **atmospheric-bg** and **glass-nav** patterns.
- Ensure all components are responsive and use the **rui-** color palette.

## ⚖️ Code of Conduct
Please be respectful and professional in all interactions. Let's build something great together.
